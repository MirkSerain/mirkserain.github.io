Commands:

Quote System

The quote system chat commands are broadcaster and moderator only. All others may use the channel point redemption option to add a quote.
capitalization matters

  • !quote add *
    !quote add this is an example
  • !quote addOther *
    !quote addOther Bomberpalmer this is a quote from another user
  • !quote random
    This will display a random quote
  • !quote #
    !quote 9
    This will display quote number 9 if it exists

Reminder System

The reminder system...

  • !reminder add *
    !reminder add buy potions
    Broadcaster, Mod, and VIP
  • !reminder list
    Broadcaster and Mod only
    This command will display all the reminders in chat and then delete them from memory. One use per stream.

Informational

These commands are usable by all members of chat and are used to display links or other information
  • !links
  • !spoilers
  • !discord
